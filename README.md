## Apa itu React?

Sebelumnya saya berpikir bahwa React adalah sebuah framework. Ternyata React adalah sebuah library javascript yang deklaratif, efisien, dan fleksibel untuk membangun antarmuka pengguna. React memungkinkan kita untuk membuat antarmuka kompleks dari kumpulan kode yang kecil dan terisolasi yang disebut â€œkomponen".
Untuk membuat komponen kita perlu mendefinisikan sebuah Class yang memanggil subkelas bernama "React.Component". Atau jika anda menggunakan Visual Studio Code, anda dapat menggunakan ekstensi ES7 Reach/Redux/GraphQL/React-Native snippets, lalu menjalankan snippet "rfc".
